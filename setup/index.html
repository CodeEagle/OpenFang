🚀 OpenFang

Agent Operating System

⚠️
需要配置 LLM API Key
OpenFang 需要至少配置一个 LLM Provider 的 API Key 才能正常运行。
1 进入应用设置
在懒猫微服中，打开 应用中心 → 找到 OpenFang → 点击 设置
2 添加环境变量
在环境变量配置中，添加你的 API Key，例如：

ANTHROPIC_API_KEY=sk-ant-xxxxx

支持的 LLM Provider：
Anthropic ANTHROPIC_API_KEY
OpenAI OPENAI_API_KEY
Google Gemini GEMINI_API_KEY
DeepSeek DEEPSEEK_API_KEY
Groq GROQ_API_KEY
OpenRouter OPENROUTER_API_KEY
3 重启应用
保存设置后，点击 重启 应用即可开始使用 OpenFang。

获取 API Key: Anthropic · OpenAI · Google
